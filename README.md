## GoTravel Mobile Application

GoTravel is a cross-platform travel planning mobile application built using React Native, designed with strong performance optimization, responsive layouts, and modern UI/UX standards. The application is developed for both Android and iOS platforms, ensuring a smooth and scalable user experience.

## Features

## Authentication & Security

User signup with OTP verification

Secure login with session-based user data

Forgot password via OTP verification

Reset password functionality

Account deletion support

## Trip Management

Plan and create personalized trips

Submit trip plans via API integration

Explore trips globally with full trip details

Save and manage favorite trips

## User Profile

Create and manage user profile

Upload and update profile images

Personalized account data management

## Navigation Structure

Tab Navigation: Home, Favorites, Search, Profile

Drawer Navigation for additional app controls

## Performance & User Experience

Optimized performance and scalable architecture

Fully responsive UI across multiple devices

Lottie animations for enhanced success and interaction screens

## Technology Stack

React Native, React Navigation, Gorhom Bottom Sheet, Axios, Async Storage, Reanimated, Gesture Handler, ViewShot, CameraRoll, Lottie Animations

## Platform Support

Android and iOS
